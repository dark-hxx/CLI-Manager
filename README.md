# CLI-Manager

一款 **轻量、高性能的 Windows 桌面应用**，用于集中管理多个开发项目的 CLI 终端，支持多 Shell（PowerShell / CMD / PWsh / WSL / Bash），解决多窗口切换、重复输入命令的痛点，提升开发工作流效率。

## 核心功能

- **统一工作区管理**：项目分组、拖拽排序、快速搜索。
- **可调侧边栏**：支持拖拽调整宽度（180-500px），自动持久化用户偏好。
- **一键启动**：支持目录内批量启动终端，支持筛选/自定义选中批量启动。
- **终端内嵌**：应用内终端面板，Tab 管理多会话。
- **终端分屏**：同一 Tab 内水平/垂直分割两个终端，分隔条可拖拽调整比例。
- **终端 Tab 拖拽排序**：终端标签支持拖拽调整顺序。
- **命令面板**：`Ctrl+P` 打开全局命令面板，模糊搜索项目/命令模板/操作，键盘导航快速执行。
- **历史会话工作区**：统一查看 Claude/Codex 历史会话，支持来源筛选、全局搜索、会话内搜索、别名/标签/收藏管理。
- **历史会话时间分组**：按 Today / Yesterday / This Week / This Month / Earlier 分组展示。
- **Diff 视图**：支持查看会话中的代码改动（Unified Diff + Codex Begin Patch），并可跳回触发消息。
- **外部终端模式**：开关控制，使用 Windows Terminal（`wt`）在一个窗口内打开多个 Tab，支持按项目 Shell 配置启动。
- **多 Shell 支持**：每个项目可独立配置 Shell 类型（PowerShell / CMD / PowerShell Core / WSL / Bash）。
- **项目配置**：独立配置路径、CLI 工具、Shell、启动命令、环境变量。
- **项目健康检查**：自动检测项目路径有效性，无效路径显示警告标识。
- **命令历史**：自动记录终端命令，支持搜索和一键重放，最大 1000 条 FIFO 管理。
- **拖拽排序**：侧边栏项目和分组支持拖拽排序，排序结果持久化。
- **右键菜单**：菜单树与终端 Tab 支持快捷操作（关闭/分屏/新建等）。
- **集中设置**：齿轮入口打开设置弹窗，整合通用偏好、终端主题、快捷键、命令模板管理。
- **终端主题**：10 套预设配色（Dracula、Nord、Monokai 等），支持跟随应用主题自动切换。
- **快捷键配置**：新建/关闭终端、切换标签、命令面板等快捷键可自定义录制。
- **历史快捷键**：`Ctrl+K` 快速打开历史会话全局搜索，历史工作区内 `Ctrl+F` 聚焦会话内搜索框。
- **命令模板**：全局或项目级模板，支持变量替换（`${projectPath}`、`${projectName}`），一键发送到终端。
- **空状态与加载态**：项目区骨架屏 + 终端空态引导，减少首次加载和空列表困惑。
- **可见通知反馈**：集成 Sonner Toast，用于关键失败场景的即时提示。
- **窗口尺寸保底**：限制最小窗口尺寸（800x500），避免极端缩放导致布局失真。

## 技术栈

- **Tauri 2.x**
- **Rust**（PTY 进程管理，多 Shell 支持）
- **React 19 + TypeScript + Vite 7**
- **xterm.js**
- **SQLite（tauri-plugin-sql）**
- **tauri-plugin-store**
- **Zustand**
- **Tailwind CSS 4**
- **@dnd-kit**（拖拽排序）

## 开发与构建

```bash
npm install
npm run tauri dev
npm run tauri build
npx tsc --noEmit
cd src-tauri && cargo check
cd src-tauri && cargo test
```
