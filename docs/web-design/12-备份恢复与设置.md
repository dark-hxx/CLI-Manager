# 备份恢复与设置

## 1. 备份概览

展示备份模式、连接状态、上次成功、待上传 Outbox、远程目录和设备快照数。WebDAV 密码留在桌面系统凭据库，Web 不能读取明文。

## 2. 快照列表

按设备分组展示时间、版本、平台、数据域、大小、哈希和状态。支持预览、下载、删除和恢复；每设备自动保留最近 10 份。

## 3. 备份范围

工作区、应用偏好、模型价格、Hook 通知设置和状态栏配置。明确排除终端标签、分屏、Workspan、滚屏、设备设置、背景图、桌宠资源、会话历史和外部 CLI 数据。

## 4. 恢复流程

1. 选择快照。
2. 查看五个数据域摘要和安全提示。
3. 勾选需要完全还原的数据域。
4. 桌面端创建本地安全快照。
5. 执行恢复，失败自动回滚。
6. 成功后提供一次撤销。

## 5. 本地 ZIP

支持由桌面端选择导出目录和导入文件。页面明确提示项目环境变量、模板、命令提示 API Key 和 Hook 凭据会按现有策略以明文进入 `snapshot.json`，应妥善保管并优先使用 HTTPS WebDAV。

## 6. 设置分类

Web 外观、语言与时区、通知、账户与安全、设备权限、供应商、Hook、备份、模型价格和关于。终端字体、终端主题、背景图、快捷键、关闭行为、硬件加速、桌宠等桌面专属配置以设备设置形式呈现。

## 7. 设置保存

即时设置立即生效；高风险和跨设备设置使用显式保存。离开未保存页面时提示。每项设置标注作用域：当前浏览器、账户、设备、项目或会话。

## 8. 安全

登录会话管理、设备信任、权限范围、二次验证、审计日志和全部退出。危险区包括撤销设备、清空 Web 数据和删除账户，必须输入确认文本。

## 9. 国际化和主题

支持 `zh-CN`、`en-US`、浅色、深色和跟随系统。主题设置提供实时预览，切换后立即应用并记忆当前浏览器偏好；跟随系统时监听系统变化。时间保持 24 小时制；自定义文本颜色需要实时对比度检查。

浅色预览采用淡蓝背景、白色卡片和蓝色主操作；深色预览采用黑灰背景、深色卡片和 terminal-green 主操作。用户也可为代码/Diff 区域独立选择浅色或深色代码主题。
