# 历史与 AI Replay

## 1. 历史列表

按 Today、Yesterday、This Week、This Month、Earlier 分组。支持来源、设备、项目、模型、收藏、标签、日期和关键字筛选；侧栏宽度在桌面端记忆。

## 2. 会话详情

页签为对话、时间线、上下文、文件变更、工具诊断、统计。消息和 Diff 双向定位；长消息折叠，代码块独立滚动。

## 3. 历史操作

收藏、标签、重命名、复制会话 ID、导出、恢复到项目、Fork 新会话、批量选择和删除。删除前展示来源文件、项目边界和影响数量；不支持的来源只读。

## 4. 消息编辑和删除

仅可编辑的用户消息显示操作；编辑产生新版本并保留审计。批量删除模式支持点击整条消息及键盘 Enter/Space，结构化工具消息不可编辑。

## 5. AI Replay

按用户 Prompt 聚合轮次，最新轮次优先。展示 AI 当前动作、用户需求、最终回复、工具、MCP、Skill、验证、文件、子任务、权限、通知和异常。详细日志支持搜索、类型筛选和原始 payload 查看。

## 6. 快照

快照详情支持查看 Diff、恢复到该快照和基于快照 Fork 分支。恢复必须显示目标设备、项目、当前变更和覆盖风险；离线时只允许查看。

## 7. Prompt Library

支持全局、项目、会话三级作用域，搜索、标签、变量、复制、插入、新建、编辑和删除。会话模板随会话生命周期清理；变量未填写时禁止发送。

## 8. 会话恢复

恢复弹窗按目录分组项目配置，搜索时自动展开匹配项。恢复前检查目标路径、CLI、供应商和设备在线状态；无法恢复时仍允许只读查看历史。

## 9. 空态与错误

无历史、索引中、同步延迟、来源目录不可读、会话损坏、内容已删除、设备离线。列表保留最近成功数据并标注新鲜度。
