# 工作台与 AI 对话

## 1. 页面结构

主工作台参考用户提供的智能协作平台布局：左侧导航与最近会话、中间对话入口、右侧新建任务与后台任务。顶部始终显示设备、项目、Worktree、分支、CLI、模型、推理等级和状态。

### 1.1 首屏空状态

- 左侧：品牌、Skills、MCP、新对话、工作空间和最近会话。
- 中间：问候语、解决项目/修复问题/头脑风暴快捷入口和大输入框。
- 右侧：新建对话、项目、代码审查、SSH 连接和后台任务等快捷卡片。
- 移动端：隐藏左右栏，把快捷入口置于问候语下方，保留固定输入框。

### 1.2 执行状态

发送 Prompt 后，中间区域切换为结构化对话；右侧快捷任务切换为执行过程、审批和变更摘要。左侧继续保留会话切换，避免执行中丢失上下文。

## 2. 新建对话

用户依次选择设备、项目或 Worktree、Claude/Codex、供应商配置、模型、推理等级和权限策略。系统在发送前检查设备在线、路径健康、CLI 可用和 Hook 状态。

## 3. 消息类型

| 类型 | 默认展示 |
|---|---|
| 用户 Prompt | 完整文本、附件和发送时间 |
| AI 进展 | 当前动作、耗时、步骤计数 |
| 工具调用 | 名称、目标、状态、结果摘要 |
| 子任务 | 代理名称、目标、进度、结果 |
| 文件变更 | 路径、增删行、Diff 入口 |
| 验证 | 命令摘要、通过/失败、首个错误 |
| 审批 | 影响范围、参数、风险和决策按钮 |
| 最终回复 | 完整回复、复制、关联变更 |
| 异常 | 原因、已完成内容、重试建议 |

## 4. 输入区

- 多行 Prompt，支持粘贴文本、图片和文件引用。
- Prompt 模板按全局、项目和会话作用域选择。
- 支持模型、推理等级、权限策略和供应商摘要。
- 运行中主按钮变为停止；停止前说明已产生的文件变更不会自动撤销。
- 离线时允许保存草稿，不允许伪装为已发送。

## 5. 审批

审批卡显示目标设备、项目、Worktree、命令或文件、风险等级和过期时间。决策为允许一次、本会话允许、拒绝；删除、覆盖、提交、推送、恢复、安装 Hook 等操作必须二次确认。

## 6. 执行时间线

按真实顺序展示 Prompt、检索、读取、工具、子任务、文件、验证、审批和最终回复。当前步骤展开，已完成步骤折叠；原始输入输出仅在“技术详情”中查看。

## 7. 会话操作

重命名、收藏、复制链接、导出摘要、停止、重新运行最后一轮、基于当前上下文新建会话、打开历史和删除。删除只影响 Web 索引还是同时删除桌面历史必须明确选择。

## 8. 状态矩阵

| 状态 | UI 行为 |
|---|---|
| 准备就绪 | 输入可用 |
| 排队 | 显示队列位置，可取消 |
| 执行中 | 展示实时步骤和停止按钮 |
| 待审批 | 固定高亮审批卡，推送通知 |
| 已完成 | 展示最终回复和变更摘要 |
| 已失败 | 展开首个错误，提供重试 |
| 已停止 | 标记停止点和已产生结果 |
| 设备离线 | 冻结时间线，允许等待或取消 |

## 9. 移动端

侧栏和过程栏改全屏抽屉；审批卡靠近输入区；Diff 默认显示文件摘要，进入独立页面查看代码。底部导航不遮挡输入框和系统安全区。

- 冷启动先展示最后一次桌面上报缓存，再原位刷新，不显示空白首页。
- 设备、项目和会话分别保存草稿与滚动位置。
- 简单选择使用 Bottom Sheet，文件、Diff、Replay 使用独立页面。
- Web Push 点击后直接进入关联设备、项目和会话。
- 协议或桌面版本不兼容时进入阻断页，不进入半可用工作台。

## 10. 主题适配

- 浅色：淡蓝灰页面背景、白色对话表面、蓝色主按钮，快捷入口使用低饱和蓝/橙/绿图标底色。
- 深色：黑灰表面和 terminal-green 主色，保持相同区域和组件尺寸。
- 代码、Diff、终端日志类内容在浅色主题下使用独立高对比代码表面，不强制跟随纯白背景。
