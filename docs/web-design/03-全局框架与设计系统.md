# 全局框架与设计系统

## 1. 信息架构

一级导航：工作台、项目、文件、Git、历史、分析、设备、设置。全局顶部栏负责设备选择、搜索、同步状态、通知和账户；上下文侧栏随模块切换。

## 2. 布局断点

| 宽度 | 布局 |
|---|---|
| ≥1440px | 导航轨 + 上下文侧栏 + 主区 + 右侧过程栏 |
| 1024–1439px | 右侧过程栏改抽屉，侧栏可折叠 |
| 768–1023px | 导航轨保留，主从页面改页签 |
| <768px | 单栏、底部导航、上下文和过程使用全屏抽屉 |

移动布局参考 LiveAgent 的同构响应式 Web：共用页面和状态，移动尺寸默认关闭侧栏。抽屉、安全区、键盘和缓存策略参考 Orca Companion App 的成熟交互，但仍使用 Web/PWA 实现。

## 3. 双主题策略

Web 端提供浅色、深色、跟随系统三种模式。两套主题共享布局、组件尺寸、状态语义和交互流程，只切换视觉 Token；主题偏好保存在当前浏览器账户设置中，首次访问默认跟随系统。

浅色主题定位为 macOS 磨砂玻璃简白风：白色与冷浅灰背景、蓝色品牌主色、低饱和功能色和大面积留白。磨砂玻璃用于导航、工具栏、抽屉、弹窗和浮动控件，主要内容保持接近不透明。深色主题延续 CLI-Manager terminal-green 风格，适合长时间开发与夜间使用。

## 5. 浅色 Token

| Token | 色值 | 用途 |
|---|---|---|
| Background | `#F3F5F8` | 页面及冷浅灰环境背景 |
| Surface | `rgba(255, 255, 255, 0.88)` | 主面板与主要内容 |
| Glass | `rgba(255, 255, 255, 0.68)` | 导航、工具栏、抽屉与浮层 |
| Card | `rgba(255, 255, 255, 0.92)` | 内容卡片 |
| Raised | `rgba(255, 255, 255, 0.96)` | 输入、代码外的高可读表面 |
| Border | `rgba(15, 23, 42, 0.08)` | 普通边界 |
| Glass Border | `rgba(255, 255, 255, 0.72)` | 磨砂玻璃高光边界 |
| Primary | `#007AFF` | 品牌、主操作、选中 |
| Success | `#2BB673` | 完成、在线 |
| Warning | `#E9A23B` | 审批、风险 |
| Danger | `#E15361` | 删除、失败、拒绝 |
| Text | `#1D1D1F` | 主文字 |
| Muted | `#6E6E73` | 次要文字 |

磨砂表面默认使用 `blur(20px) saturate(140%)`、细高光边框和轻柔阴影；不支持 `backdrop-filter` 时回退为接近不透明的白色表面。主要内容卡片、表格、代码、Diff 和日志不得直接使用低透明玻璃层。蓝、绿、橙只用于状态和入口图标，不形成大面积彩色背景。

## 6. 字体与间距

- UI：Inter、Noto Sans SC、系统无衬线。
- 代码：JetBrains Mono、Cascadia Code、等宽兜底。
- 正文 14px，辅助 12px，标题 16/20/24px。
- 4px 基础网格；页面间距 24px，卡片 16px，紧凑行 8–12px。

## 7. 组件清单

按钮、图标按钮、分段控件、下拉菜单、搜索框、标签、状态点、卡片、数据表、树、时间线、Diff、代码块、审批卡、Toast、Dialog、Drawer、Tooltip、Skeleton、Empty State、Error State、Command Palette。

## 8. 交互状态

每个控件必须覆盖默认、Hover、Pressed、Focus、Disabled、Loading、Success、Warning、Error。Hover 只改变颜色、边框或阴影，不使用缩放导致布局抖动。

## 9. 图标与图表

- 统一使用 Lucide 风格线性图标，不使用 Emoji 充当功能图标。
- 趋势用折线/面积，排行用横向条形，构成用环形，时间活跃用热力图，效率用散点。
- 图表提供 Tooltip、图例显隐、键盘焦点和数据表入口。

## 10. 无障碍

- 正文对比度至少 4.5:1。
- 所有操作可键盘完成，焦点环清晰可见。
- 点击目标至少 44×44px，桌面紧凑表格操作区除外但不得小于 32px。
- 状态不能只用颜色表达，必须有图标或文字。
- 动画遵守 `prefers-reduced-motion`。
- 移动抽屉支持背景滚动锁定、返回键关闭和焦点恢复。
- 输入区使用安全区和可视视口高度，软键盘弹出时不得遮挡发送按钮。

## 11. 国际化

所有文案定义中英文键；时间统一 24 小时制。长英文、长项目名和 Windows/WSL 路径必须支持省略及 Tooltip，不得撑破布局。
