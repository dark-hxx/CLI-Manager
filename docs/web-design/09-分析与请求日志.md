# 分析与请求日志

## 1. 全局筛选

设备、项目、来源、模型、日期范围和时区。快捷范围为今天、近 7 天、近 30 天、本月、本年和自定义。筛选同步到 URL，支持分享只读视图。

## 2. 指标

会话数、消息数、输入 Token、输出 Token、缓存命中、缓存写入、总 Token、预估费用、在线设备和未定价 Token。指标卡展示环比和数据更新时间。

## 3. 图表

Token/费用趋势、Token 构成、项目排行、模型占比、来源对比、30 天热力图、24 小时分布和项目效率散点。点击日期下钻会话，点击项目或模型应用筛选。

## 4. 请求日志

列包含时间、来源、项目、模型、会话、输入、输出、缓存、总量、费用和异常标记。支持关键字、快捷筛选、列显隐、排序、分页、复制会话 ID 和跳转历史。

## 5. 数据口径

- 费用使用 CLI-Manager 当前模型价格计算。
- 未匹配价格的 Token 单列统计，不伪造费用。
- 请求日志只保存 usage 元数据，不保存 Prompt、响应正文、密钥、伪 HTTP 状态或伪耗时。
- Worktree 今日项目用量按父项目与 active Worktree 去重聚合。

## 6. 图表交互和无障碍

Hover 展示精确值；图例可显隐；键盘可移动焦点；每张图提供数据表入口。颜色之外使用线型、标记和文字区分类别。

## 7. 空态与性能

无数据、同步中、部分设备离线、统计失败、未定价。大范围查询显示进度和取消入口；图表使用聚合数据，不向浏览器加载全部请求日志。

## 8. 导出

支持导出当前筛选 CSV 和图表 PNG。导出前显示范围、行数和脱敏规则；移动端只保留分享与后台生成下载链接。
