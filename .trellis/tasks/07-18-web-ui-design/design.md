## Visual Direction

- Dark developer-tool interface.
- Existing terminal-green palette.
- Surface layering instead of dense separator lines.
- Structured conversation as the primary workspace.
- Real-time execution timeline and approval actions remain visible without exposing a terminal.

## Layouts

- Desktop workbench: 1600×1000.
- Desktop analytics: 1600×1000.
- Mobile conversation: 430×932.

## Constraints

- SVG only; no product implementation.
- No external font or image dependency.
- Use system font stacks and vector primitives.
- Keep Chinese and English localization expansion in mind.
